Resultados del Agente de Aprendizaje por Refuerzo

🧠 ¿Cómo se entrenó el modelo?

Se entrenó un agente usando Q-Learning con datos reales de clientes de una empresa de telecomunicaciones. El agente aprende a asociar perfiles de clientes con la acción que maximiza la retención:

  • ✔️ Se usó una matriz Q (`q_table`) para representar el conocimiento aprendido.
  • ✔️ Se simularon {{ total_episodes }} episodios con acciones como "ofrecer descuento", "enviar encuesta" o "no hacer nada".
  • ✔️ Las recompensas se asignaron según si el cliente abandonaba o no el servicio.
  • ✔️ Se utilizó escalado y codificación de variables para normalizar los datos.
